# Douglas Portfolio

Bilingual portfolio for Douglas Queiroz, focused on backend engineering, product platforms, analytics APIs, and AI-enabled workflows.

## What changed

- Replaced the old hardcoded project model with a bilingual case-study schema
- Added first-class PT/EN switching persisted in local storage
- Separated featured case studies from in-progress work
- Removed outdated Java positioning and private repository links from the UI
- Reframed project pages around problem, solution, architecture, and impact

## Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4
- Framer Motion
- shadcn/ui

## Local development

```bash
npm install
npm run dev
```

The app runs on [http://localhost:3002](http://localhost:3002).

## Validation

```bash
npm run lint
npm run build
```
